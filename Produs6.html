Capsule compatibile Bialetti® marca La Capsuleria® pentru toate modelele de espressoare de cafea Bialetti®*. Orzul este un produs versatil in bucatarie si bun la orice varsta.Nu contine cofeina si de aceea este potrivit pentru copii dar si pentru oamenii in varsta.Aceasta bautura este obtinuta din prajirea si macinarea orzului si pastreaza toate beneficiile acestei cereale. Contine carbohidrati, proteine, fibre si minerale si este foarte utila pentru a incepe o zi plina de energie. Cei mai importanti nutrienti pe care aceasta cereala ti le poate oferi sunt potaziu, fosfor si vitamina B3.Orzul solubil are foarte multe virtuti: este diuretic, digestiv, antioxidant si antiinflamator. Ajuta la bunastarea inimii si a vaselor de sange si contribuie la diminuarea nivelului de colesterol din sange. Singurul sau dezavantaj este de a avea gluten, deci nu poate fi consumat de cei intoleranti.Pe langa faptul de a fi gustoasa, este o bautura curativa si eficienta in numeroase ocazii. Este indicata femeilor gravide pentru a favoriza productia de lapte matern si este util pentru a prevenii osteoporoza. Deoarece nu este o substanta energizanta poate fi consumata si de cei tineri pentru a favoriza capacitatiile cognitive, memoria si concentratia.O bautura pe baza de orz este foarte buna pentru a stimula functionalitatiile intestinului si pentru a reduce anumite tulburari, amelioreaza durerile abdominale si este foarte util pentru persoanele in convalesceta.
